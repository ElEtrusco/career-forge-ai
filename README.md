# Career Forge AI

Career Forge AI es una plataforma impulsada por inteligencia artificial que ayuda a optimizar la búsqueda de empleo.

## Funcionalidades

- Análisis inteligente del CV.
- Optimización ATS.
- Búsqueda automática de ofertas.
- Cálculo de coincidencia.
- Generación de CV personalizados.
- Cartas de presentación automáticas.
- Seguimiento de candidaturas.
- Exportación a Excel.
- Dashboard.

## Tecnologías

- Python
- FastAPI
- Next.js
- PostgreSQL
- Docker
- OpenAI API
- Cloudflare
- GitHub Actions
